БОНУС · Как опубликовать сайт бесплатно
GitHub Pages: сайт за 10 минут
1
Зарегистрироваться на GitHub
Зайди на github.com → нажми «Sign up» → введи email, придумай логин и пароль → подтверди email. Это бесплатно.
2
Создать репозиторий
На главной нажми зелёную кнопку «New» → придумай название (например my-site) → выбери «Public» → нажми «Create repository».
3
Сгенерировать сайт в нейросети
Открой claude.ai или chatgpt.com → напиши: «Сделай одностраничный HTML-сайт о ...» → скопируй весь код → сохрани как файл index.html.
4
Загрузить файл в репозиторий
Открой свой репозиторий → нажми «Add file» → «Upload files» → перетащи index.html → нажми «Commit changes». Файл должен называться именно index.html — это важно.
5
Включить GitHub Pages
В репозитории → вкладка Settings → раздел Pages слева → в поле «Branch» выбери main → нажми Save. Подождать 1–2 минуты.
6
Открыть сайт
Ссылка появится в том же разделе Pages: твой-логин.github.io/my-site. Поделись ею — она работает у всех без регистрации.
