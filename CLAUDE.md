## Плейсхолдеры — нужно заменить реальными значениями

Найди через `Ctrl+H` в VS Code текст `[ ]` и замени на свои цифры.

### Слайд 01 — Тендеры (`#s1`)
- `[ ]x` — во сколько раз быстрее первичный анализ тендера
- `[ ]%` — на сколько снижается риск пропустить опасное условие
- `[ ] ч` — сколько часов экономит каждый тендер

### Слайд 02 — Блог ЦУМ (`#s2`)
- `[ ]` — сколько статей в месяц обеспечено AI-изображениями
- `[ ]x` — экономия по сравнению с дизайнером

### Слайд 03 — Чат-бот (`#s3`)
- `[ ]` — сколько контактов собрано через бота

### Слайд 04 — Аудиты (`#s4`)
- `[ ] ч` — сколько часов экономит один аудит
- `[ ]x` — рост скорости подготовки аудита

### Слайд 05 — Контент (`#s5`)
- `[ ]%` — экономия времени на контент-план
- `[ ]x` — рост скорости подготовки сценария
- `[ ]` — постов в месяц с AI-помощью

## Навигация в презентации
- Клавиши `←` `→` — переключение слайдов
- Клики по точкам в шапке — прыжок на нужный слайд
- Кнопки `←` `→` в правом углу шапки
- Прогресс-бар внизу показывает текущую позицию

---

## Как добавить новый слайд

1. Скопируй блок любого существующего слайда (`<div class="slide" id="sN">`)
2. Присвой новый ID: `s7`, `s8` и т.д.
3. Обнови константу в JS: `const TOTAL = 8;`
4. Добавь метку в массив: `const labels = [..., 'Название слайда'];`

---

## Как изменить цвета акцентов

В блоке `:root` в начале `<style>`:

```css
--accent: #e8c97a;   /* золотой — основной акцент */
--accent2: #4a9eff;  /* синий */
--green: #4fc08d;    /* зелёный */
--red: #e05c5c;      /* красный */
```

## Классы рамок для блоков `.box`

| Класс | Цвет | Когда использовать |
|-------|------|--------------------|
| `.b-gold` | Золотой | Основное, позитивное |
| `.b-blue` | Синий | Техническое, процесс |
| `.b-green` | Зелёный | Результат, решение |
| `.b-red` | Красный | Проблема, было |
