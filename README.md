# ИИ в работе отдела

Интерактивная HTML-презентация для собрания руководителей компании.  
Тема: как отдел использует искусственный интеллект в повседневной работе.

## Слайды

| # | Тема | Инструменты |
|---|------|-------------|
| 0 | Титульный слайд | — |
| 1 | Работа с тендерами | Claude |
| 2 | Генерация изображений для блога ЦУМ | ChatGPT / DALL·E |
| 3 | Чат-бот для вебинаров | Perplexity |
| 4 | Предпродажные аудиты | Claude Cowork + ChatGPT |
| 5 | Сценарии, контент-планы, посты | Claude + ChatGPT |
| 6 | Сервис SEO-отчётности | Claude + Claude Code + GPT API |

## Навигация

- Клавиши `←` `→` — переключение слайдов
- Клики по точкам в шапке — переход на нужный слайд
- Кнопки `←` `→` в правом углу шапки
- Прогресс-бар внизу показывает текущую позицию

## Структура

```
index.html          — основной файл презентации
assets/
  css/style.css     — все стили
  js/main.js        — логика навигации
```
